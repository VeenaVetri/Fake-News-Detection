# Fake-News-Detection
The Fake News Detection System is a machine learning project aimed at classifying news articles as authentic or fabricated. Leveraging LSTM neural networks and state-of-the-art natural language processing techniques, this system provides an effective solution to combat the spread of misinformation in the digital information landscape.

## Dataset
https://drive.google.com/drive/folders/1U9EYESizwThweYaZ5yK_Z_77df81PXOV?usp=sharing

## Features
- Utilizes LSTM neural networks for accurate classification of news articles.
- Incorporates advanced natural language processing techniques to analyze textual content.
- Achieves a high accuracy of 90.35% on the test dataset, demonstrating robust performance.
- Empowers users to make well-informed decisions by distinguishing between trustworthy and deceptive sources.
- Offers versatile deployment options for media professionals, fact-checking organizations, and online platforms.

## Results
- **Accuracy**: 90.35%
